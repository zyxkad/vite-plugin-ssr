.env Files

Vite automatically makes environment variables defined in .env files available at {"BASE_URL":"/","MODE":"production","DEV":false,"PROD":true}, see Vite's docs about .env files.

Note that:

  • It only works for environment variables that are prefixed with VITE_ (for security reasons).
  • It only works for files processed by Vite (all .page.js/.page.server.js/.page.client.js/.page.route.js files and all their imports). For files not processed by Vite (e.g. Express.js server) we can load our .env file ourself:
    // server.js
if (!isProduction) {
  require('dotenv').config() // npm install dotenv
}
